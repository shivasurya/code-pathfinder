### Variable Declaration Entity <Badge text="Entity" variant="tip" size="small" />

Variable declaration entity represents a variable declaration in the source code. It contains information about the variable name,
visibility, return type, and other attributes.

```java
<variabledatatype> <name> = <variablevalue>;
```

### Attributes

| Entity                    | Description                                              |
|---------------------------|----------------------------------------------------------|
| `name`                    | Entity's name                                            |
| `visibility`              | Entity's visibility (public, private, protected)         |
| `variabledatatype`        | Entity's variable data type                              |
| `variablevalue`           | Entity's variable value                                  |
| `has_access`              | Check if the entity is being accessed based on scope     |
| `scope`                   | Entity's scope (local, field)                            |
| `is_java_source`          | Is the entity a Java source code                         |
