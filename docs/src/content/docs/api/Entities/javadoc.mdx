### JavaDoc Comment Entity <Badge text="Entity" variant="tip" size="small" />

JavaDoc comment entity represents a JavaDoc comment in the source code. It contains information including comment, author, see, version
and other attributes.

```java
/**
 * <multi line description>
 * @author <AUTHOR NAME>
 * @see <link>
 * @version <version>
 * @param <name> <type> <description>
 * @return <type> <description>
 * @throws <type> <description>
 * @exception <type> <description>
 * @since <version>
 * @deprecated <description>
*/
```

### Attributes

| Entity                    | Description                                              |
|---------------------------|----------------------------------------------------------|
| `comment_author`          | Author of the block comment. (example: @author shiva)    |
| `comment_see`             | See references of the block comment                      |
| `comment_since`           | Since references of the block comment (@since 01-01-2010)|
| `comment_version`         | Version references of the block comment                  |
| `comment_param`           | Param references of the block comment                    |
| `comment_throws`          | Throws references of the block comment                   |
| `is_java_source`          | Is the entity a Java source code                         |
