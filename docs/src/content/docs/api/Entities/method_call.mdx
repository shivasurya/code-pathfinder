### Method Call Entity <Badge text="Entity" variant="tip" size="small" />

Method call entity represents a method call in the source code. It contains information about the method name, arguments, and other attributes.

```java
<name>(<parameter>)
```

### Attributes

| Entity                    | Description                                              |
|---------------------------|----------------------------------------------------------|
| `name`                    | Entity's name                                            |
| `argumentname`            | Entity's parameter name                                  |
| `is_java_source`          | Is the entity a Java source code                         |
